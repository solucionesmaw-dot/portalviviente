# **App Name**: Portal de Abundancia

## Core Features:

- Sección Hero: Sección hero visualmente atractiva con título, subtítulo, imagen/fondo (portal, galaxia, energía de luz) y botones destacados de 'Reserva tu lugar' y 'Más información'.
- Descripción del Evento: Resalta el propósito del evento y el público objetivo.
- Visualización de Beneficios: Lista visualmente atractiva de beneficios (abundancia material, éxito profesional, realización personal, relaciones, bienestar) con íconos representativos.
- Desglose de Inclusiones: Descripción clara de lo que reciben los participantes (meditaciones, activaciones, trabajo energético, grupo de WhatsApp).
- Detalle de la Experiencia Nocturna: Explica el formato de cada noche.
- Precios y Promociones: Sección clara que muestra las opciones de precios para individuos y parejas.

## Style Guidelines:

- Color primario: Violeta pastel (#E6CCFF) para evocar misterio y espiritualidad.
- Color de fondo: Gris claro pastel (#E0E0E0) para enfatizar el tema místico.
- Color de acento: Dorado pastel (#FFECB3) para representar la abundancia y la energía divina.
- Fuente del encabezado: 'Playfair', una serif moderna con un toque elegante, moderno y de alta gama.
- Fuente del cuerpo: 'PT Sans', una sans-serif humanista para complementar el encabezado.
- Utilizar íconos místicos y elegantes para los beneficios (estrellas, rayos de luz, etc.).
- Animaciones sutiles con efectos de luz o partículas de estrellas al pasar el cursor sobre los botones.
